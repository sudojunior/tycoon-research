# Airline Pilot

## Data Input

### Required

- Timestamp  
  *required for OS or Beta if you don't submit when you complete route*
  - Date (mm/dd/yyyy)
  - Time (h:mm:ss {AM/PM})
- Server
- Aircraft
- Departure (Airport)
- Arrival (Airport)
- Earnings
  - Money
  - EXP

### Optional

- Player ID (for [Driver Credit](../project/DriverCredit.md))
- Departure (Terminal)
- Arrival (Terminal)
- Flight
  - Distance (m)
  - Duration (h:mm:ss)
- Earnings
  - EXP
- Notes  
  *If EXP tokens were used, please add it here.*
