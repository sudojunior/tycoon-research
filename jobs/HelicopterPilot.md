# Helicopter Pilot

## Data Input

### Required

- Timestamp  
  _required for OS or Beta if you don't submit when you complete route_
  - Date (mm/dd/yyyy)
  - Time (h:mm:ss {AM/PM})
- Server
- Helicopter
- Journey
  - Departure
  - Arrival
- Earnings
  - Money
  - EXP
- Tokens (Used)
- [Boosts](../project/Common.md#Boosts)

### Optional

- Player ID (for [Driver Credit](../project/DriverCredit.md))
- Journey (Duration)
- Tokens (Earned)
- Notes
